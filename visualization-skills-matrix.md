## Selection Recommendations by Role

### Product Marketing (Brian's Primary Use)

**Top 3 Skills:**
1. **ciq-visual-design** - Product announcements, feature graphics, market positioning
2. **rlc-ai-visuals** - RLC-AI specific marketing and technical demos
3. **d3-visualizations** - Performance comparisons, data-driven sales materials

**Workflow:**
- Use `ciq-visual-design` for 80% of product marketing needs
- Use `rlc-ai-visuals` when specifically targeting AI/HPC infrastructure narrative

## Conclusion

**For most CIQ product marketing work**, start with these defaults:
- **Static graphics** → `ciq-visual-design`
- **RLC-AI content** → `rlc-ai-visuals`
- **Data visualization** → `d3-visualizations`
- **Presentations** → `cinematic-scroll-deck`

**Advanced users** can layer these:
- Use `emotional-viz-design-system` for custom narrative structures
- Combine `ciq-visual-design` + `d3-visualizations` for data-rich marketing
- Use `canvas-cinematic-presentations` for high-impact training

**Quick decision rule:**
```
IF (RLC-AI specific) → rlc-ai-visuals
ELSE IF (data/charts) → d3-visualizations
ELSE IF (CIQ branded graphics) → ciq-visual-design
ELSE IF (narrative presentation) → cinematic-scroll-deck
ELSE IF (conceptual metaphor) → emotional-viz-design-system
```
